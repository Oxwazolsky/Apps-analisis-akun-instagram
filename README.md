# 📊 InstaInsight - Instagram Account Analysis App
InstaInsight adalah aplikasi web sederhana yang digunakan untuk menganalisis akun Instagram secara visual.  
Aplikasi ini menampilkan informasi analisis akun dengan tampilan dashboard yang modern dan responsif.

Project ini dibuat sebagai latihan pengembangan web menggunakan HTML, CSS, dan JavaScript.

---

## 🚀 Fitur Utama
- 📈 Dashboard analisis akun
- 🎨 Tampilan modern menggunakan Tailwind CSS
- 📊 Visualisasi informasi akun
- 🌙 Dark mode interface
- ⚡ Interface ringan dan responsif

---

## 🛠️ Teknologi yang Digunakan
Project ini dibangun menggunakan teknologi berikut:

- **HTML5**
- **CSS3**
- **JavaScript**
- **Tailwind CSS**
- **Font Awesome**
